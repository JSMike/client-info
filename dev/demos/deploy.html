What is an operating system?

The software your computer uses to run all the different applications you use. Most people use Microsoft Windows.

What is a browser?

The software you use to view a website. There are many different browsers, each with many versions. The most common are Microsoft's Internet Explorer, Google's Chrome, and Mozilla's Firefox.

But this used to work on my device...

In order to support newer browsers sometimes we have to stop supporting older browsers. You may find that the website still works in these older browsers, but we cannot guarantee that.

Clear browser cache

Windows

Chrome
  1. In the browser bar, enter: `chrome://settings/clearBrowserData`
  2. From the `Obliterate the following items from:` drop-down menu, select `the beginning of time`.
  3. Select the items you want to clear (e.g., `Browsing history`, `Download history`, `Cookies and other site and plug-in data`, `Cached images and files` )
  4. Click the `Clear browsing data` button.
Firefox
  1. Press `Alt` to make the menu bar visible.
  2. Click `Tools` then `Options`
  3. Select the `Privacy` tab
  4. From the `Time range to clear:` select `Everything`
  5. Click the down arrow next to `Details` and check all items
  6. Click `Clear Now`
Internet Explorer
  1. Press `Alt` to make the menu bar visible.
  2. From the `Tools` menu, select `Delete browsing history...`.
  3. Deselect `Preserve Favorites website data`, and select all other options
  4. Click `Delete`.

Macintosh

Safari
  1. From the `Safari` menu, select `Reset Safari... `.
  2. From the menu, select the items you want to reset, and then click `Reset`. As of Safari 5.1, `Remove all website data` covers both cookies and cache.

Android

Android Browser
  1. Start your browser.
  2. Tap `Menu`, and then tap `More`.
  3. Select `Settings`.
  4. Under `Privacy settings`, select `Clear cache`, `Clear history`, and `Clear all cookie data`
  5. Tap `OK`

iOS

Safari iOS 7
  1. On the home screen click on `Settings`
  2. Scroll down and tap on `Safari`
  3. Scroll down and tap `Clear Cookies and Data`
Safari iOS 6
  1. On the home screen click on `Settings`
  2. Scroll down and tap on `Safari`
  3. Scroll down and tap `Clear Cache`

Enable Javascript

Windows

Chrome
  1. In the browser bar, enter: `chrome://settings/content`
  2. Look for the `JavaScript` section and select `Allow all sites to run JavaScript (recommended)`.
  3. Click on the `Done` button to close dialog.
  4. Close the `Settings` tab.
  5. Refresh the page you were trying to view
Firefox
  1. In the address bar, type about:config and press Enter
  2. If prompted, Click `I`ll be careful, I promise`
  3. In the search box, search for javascript.enabled
  4. Toggle the `javascript.enabled` preference (right-click and select `Toggle` or double-click the preference) to change the value from `false` to `true`.
Internet Explorer 9/10/11
  1. Press `Alt` to make the menu bar visible.
  2. Click `Tools` menu and select `Internet Options`.
  3. In the `Internet Options` window select the `Security` tab.
  4. On the `Security` tab click on the `Custom level...` button.
  5. When the `Security Settings - Internet Zone` dialog window opens, look for the `Scripting` section.
  6. In the `Active Scripting` item select `Enable`.
  7. When the `Warning!` window pops out asking `Are you sure you want to change the settings for this zone?` select `Yes`.
  8. In the `Internet Options` window click on the `OK` button to close it.
  9. Click on the `Refresh` button of the web browser to refresh the page.
Internet Explorer 8
  1. On web browser menu click `Tools` and select `Internet Options`.
  2. In the `Internet Options` window select the `Security` tab.
  3. On the `Security` tab click on the `Custom level...` button.
  4. When the `Security Settings - Internet Zone` dialog window opens, look for the `Scripting` section.
  5. In the `Active Scripting` item select `Enable`.
  6. When the `Warning!` window pops out asking `Are you sure you want to change the settings for this zone?` select `Yes`.
  7. In the `Internet Options` window click on the`OK` button to close it.
  8. Click on the `Refresh` button of the web browser to refresh the page.

Macintosh

Firefox
  1. Select the `Firefox` menu item from the `Apple/System bar` at the top of the screen.
  2. From the drop-down menu, select `Preferences...`
  3. Select the `Content` icon/tab at the top on the window.
  4. Then, check the `Enable JavaScript` checkbox.
  5. `Close` the Options window to save your changes.
  6. Finally, `refresh` your browser.
Safari
  1. Select the `Safari` menu item from the `Apple/System bar` at the top of the screen.
  2. From the drop-down menu, select `Preferences`.
  3. Select the `Security` icon/tab at the top of the window.
  4. Then, check the `Enable JavaScript` checkbox under the `Web content` category.
  5. `Close` the window to save your changes.
  6. Finally, `refresh` your browser.
Chrome
  1. Select the `Chrome` menu item from the Apple/System bar at the top of the screen.
  2. From the drop-down menu, select `Preferences...`
  3. On the left, select `Settings`
  4. At the bottom of the page, click `Show advanced settings…`
  5. Under Privacy, select the `Content settings…` button
  6. Finally, under the JavaScript heading, select the `Allow all sites to run JavaScript` radio button

Android

  1. Select Browser or Internet to open the web browser
  2. Select Menu
  3. Select More then Settings
  4. Finally select Enable JavaScript

iOS

  1. On the home screen click on `Settings`
  2. Scroll down and tap on `Safari`
  3. Scroll down and tap `Advance`
  4. Ensure `JavaScript` is enabled

Update your operating system

While updating your operating system may not always be possible, it is often the first step at maintaining support for new software and websites.

On desktops there is often cost involved to get the newest version and you may need a professional to install it. It is most common that when your operating system needs upgraded its time to purchase a new computer.

In most cases, on hand held devices such as tablets or phones, the newest operating system is free, and easy to install.

Update your browser

While our website might run without major issues in older browser versions, we do not actively test it in them and generally do not fix bugs that may appear in them.

Updating your browser is free and the best way to:

  • Improve security against viruses, trojans, and malware
  • Keep the browser starting fast
  • Load websites faster
  • Keep websites working and displaying properly
  • Get support from the websites you use

Enable Cookies

Windows

Internet Explorer
  1. Press `Alt` to make the menu bar visible.
  2. Select the `Tools->Internet Options`
  3. Open the `Privacy` tab
  4. You can then adjust the slider to `Medium`
Firefox
  1. Press `Alt` to make the menu bar visible.
  2. Use the `Tools->Options`
  3. Choose the `Privacy` tab
  4. Set the `Firefox will:` option to `Remember history`
Chrome
  1. In the browser bar, enter: `chrome://settings/content`
  2. Set `Cookies` to `Allow local data to be set (recommemded)
  3. Click `Done`

Macintosh

Chrome
  1. In the browser bar, enter: `chrome://settings/content`
  2. Set `Cookies` to `Allow local data to be set (recommemded)
  3. Click `Done`
Firefox
  1. Select the `Firefox` menu item from the `Apple/System bar` at the top of the screen.
  2. From the drop-down menu, select `Preferences...`
  3. Choose the `Privacy` tab
  4. Set the `Firefox will:` option to `Remember history`
Safari
  1. Select the `Safari` menu item from the `Apple/System bar` at the top of the screen
  2. From the drop-down menu, select `Preferences`
  3. Open the `Privacy` tab
  4. Set `Block cookies and other website data` to `From third parties and advertisers

iOS

  1. From the home screen tap the `Settings` icon
  2. Then choose `Safari`
  3. Find the `Block Cookies` option
  4. Set to `From third parties and advertisers`

Android

  1. Tap the `Internet` icon to start the browser
  2. Then press the phone`s menu button
  3. From the list of options, choose `More`
  4. Then select `Settings`
  5. Scroll down the resulting list and check the `Accept Cookies` item

My new browser is not yet supported / My browser is supported, but says unsupported

Occasionally a browser can get mis-diagnosed. This can occur for a number of reasons, including extensions that are installed. If you would like you can report it here. Otherwise you can just ignore this mis-diagnosis.

If you are using the latest release of your browser, please be aware that there may be a delay in our support of the new browser release while we conduct a review to ensure it meets our security standards. We recommend that you download and install a supported browser until the new version is officially supported.
